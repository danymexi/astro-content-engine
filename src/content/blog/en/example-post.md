## Welcome to Your Content Engine

This is an example post that demonstrates the content structure. You can:

1. Write posts manually in `src/content/blog/en/` and `src/content/blog/it/`
2. Use the generator: `node scripts/generate-article.mjs`
3. Let GitHub Actions publish automatically

## How to Customize

Edit `scripts/topic-queue.json` to define your content plan, then run the generator.
